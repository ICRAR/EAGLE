Graph Updater

A tool to update Logical Graphs to the latest version of the EAGLE Logical Graph file format. More info...

Steps:

  • Select a source repository.
  • Click "Fetch Logical Graphs" to load all Logical Graphs from the source repository into memory.
  • Review the list of Logical Graphs, deselect any files that don't require updating.
  • Click "Update Logical Graphs" to update them in memory.
  • Review the updated Logical Graphs, deselect any files that didn't update correctly.
  • Select a destination repository.
  • Click "Push" to write selected Logical Graphs to the destination repository.

Typically, the destination repository is a branch of the source repository.

Once the updated Logical Graphs have been verified by the user, we leave it to the user to merge the new branch outside of EAGLE.
